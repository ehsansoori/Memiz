# Memiz API

An extensible ASP.NET Core Web API for generating AI-powered flashcards and learning cards.

## 🚀 Features

- AI-powered flashcard generation
- Multi-language support (English, Persian, and more)
- Pluggable AI providers (OpenAI, ChatGPT, etc.)
- Pluggable dictionary providers
- Template-based card generation
- Clean and extensible architecture
- RESTful API design
- Swagger/OpenAPI documentation

## 🛠 Tech Stack

- ASP.NET Core Web API
- C#
- .NET
- Swagger (Swashbuckle)
- OpenAI API

## 📁 Project Structure

- Controllers
- DTOs
- Services
- Providers
- Interfaces
- Factories
- Templates
- Models

## 🎯 Purpose

Memiz is designed to help developers build intelligent flashcard and vocabulary-learning applications by combining AI-generated content with structured card templates.
